# 项目简介

## 核心功能
- 用户管理
- 宠物信息展示
- 寄养与美容服务
- 订单处理
- 动态发布
- 知识分享
- 宠物用品销售
- 社区互动

## 适用场景
- 宠物交易市场
- 宠物服务平台
- 宠物相关业务管理

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
